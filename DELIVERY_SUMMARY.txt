                       HOW TO USE - QUICK START

INSTALLATION (5 minutes):

1. Install Python dependencies:
   pip install -r backend/requirements.txt
   pip install -r frontend/requirements.txt

2. Set up environment:
   echo "GROQ_API_KEY=your_key_here" > .env
   (Get free key at https://console.groq.com)

3. Start backend (Terminal 1):
   python -m uvicorn backend.main:app --reload

4. Start frontend (Terminal 2):
   streamlit run frontend/streamlit_app.py

5. Open browser:
   http://localhost:8501

6. Test everything (Terminal 3):
   python scripts/test_all_endpoints.py

7. Verify project:
   python scripts/final_check.py

================================================================================
                       HOW IT WORKS - USER FLOW
================================================================================

1. USER PASTES ARTICLE
   └─ Article text in sidebar
   └─ Select persona (investor/student/founder/journalist)

2. SYSTEM PROCESSES ARTICLE
   └─ Cleans text (removes HTML, URLs, normalizes)
   └─ Chunks into semantic pieces
   └─ Generates embeddings (384D vectors)
   └─ Searches database for similar articles

3. AI GENERATES BRIEFING
   └─ 4 questions answered: what/why/who/next
   └─ Personalized for selected persona
   └─ Related articles displayed
   └─ Response time: <10 seconds

4. USER CAN EXPLORE
   └─ Read briefing
   └─ Click related articles
   └─ Ask follow-up questions
   └─ See citations and sources

5. Q&A WITH CITATIONS
   └─ User asks question
   └─ System retrieves relevant articles
   └─ LLM generates answer based on sources
   └─ Citations show sources and quotes
   └─ Response time: <10 seconds
